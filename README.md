# Todo List Application

## Project Overview
The Todo List Application is a simple and efficient task management tool that helps users organize their daily activities. It allows users to add, remove, and clear tasks, providing a clean and intuitive interface for managing todos. This project focuses on strengthening core frontend development concepts and clean UI practices.

---

## Features
- Add new tasks
- Remove individual tasks
- Clear all tasks at once
- Simple and user-friendly interface
- Responsive design

---

## Tech Stack
- **Frontend:** HTML, CSS, JavaScript
- **Tools:** Git, GitHub

---

## How It Works
1. Users can add tasks using the input field.
2. Each task is displayed in a list format.
3. Tasks can be removed individually when completed.
4. All tasks can be cleared with a single action.
5. The application updates the UI dynamically based on user actions.
